Note:

How to make it works?

1. Check your environment settings

ruby programming language is required to run this program.

check your ruby version, open your terminal, run:
> ruby -v
If you can't see output similar like below:

ruby 2.5.1p57 (2018-03-29 revision 63029) [x86_64-darwin17]

You probably have to install ruby first.
Check the link below to install RVM and Ruby
https://rvm.io/

2. Easy to Install:
In your terminal, run:
./install.sh

Then, you are good to use this tool.

3. Try Examples
example:
1. Get help information
? -h

2. Get example cases
? -e

3. Valid input
? 1/3+1/4
= 7/12
